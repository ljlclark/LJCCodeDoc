DataManager Add Overview

The DbRequest.Columns object includes the definitions for the columns to be added to the base table.

The DbRequest.Columns include all the columns from the BaseDefinition by default. It includes only the columns contained in the optional propertyNames parameter if provided. It uses the columns in the ChangedNames object if it exists and the propertyNames parameter is null.

The propertyNames must not include calculated or join columns. This will cause an error.

The DbRequest.Columns do not include AutoIncrement columns or columns with null values. To include a null column value, set the column value to "-null".

The lookup Key columns must be available in the BaseDefinition object to be included.

The lookup key columns do not use keys with null values or invalid dates.

The result contains the database assigned column values.
