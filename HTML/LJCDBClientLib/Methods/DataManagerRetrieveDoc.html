DataManager Retrieve Overview

The Retrieve() method must have keyColumns or filters to limit the returned records or the request will not be executed.

The DbRequest.Columns object includes the definitions for the columns to be retrieved from the base table.

The DbRequest.Columns include all the columns from the BaseDefinition by default. It only includes the columns contained in the optional propertyNames parameter if provided.

The keyColumn.ColumnName must be available in the BaseDefinition object or the join.Columns parameter to be included.

The keyColumns do not use keys with null values. To include a null column value, set the column value to "'-null'".

The Retrieve() method returns a result that only includes the first record from the query result.
