Remarks

Creates a private LJCDataDetailDAL.DataDetailData object to provide simplified interaction with the database.

Calls DataDetailData.GetControlDetail() to create a private LJCDataDetailDAL.ControlDetail object to contain the dialog configuration retrieved from the database. A default configuration is created if a database configuration is not found.
