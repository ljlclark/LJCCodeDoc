Assembly LJCDBDataAccess

Message Data Access Library

The Message Data Access library contains classes, which receive DbRequest Request objects and performs the requested data operations. The data is returned to the calling program as a DbResult object.

The primary class is DbDataAccess.

The Message Data Access Library classes use LJCDataAccess to communicate with the database.

The Message Data Access library is part of the LJCDBServiceLib solution.
