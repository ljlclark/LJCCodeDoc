public DbColumn LJCAddPropertyAs(String propertyName, String caption = null, String renameAs = null, String dataTypeName = String)

Creates the DbColumn from the supplied values and adds to the collection. (R)

Remarks

Accepts the property name and sets the column name the same as the property name. If the renameAs value is null, then it is also set the same as the property name.
