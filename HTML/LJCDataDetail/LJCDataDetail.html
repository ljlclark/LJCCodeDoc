Assembly LJCDataDetail

The LJCDataDetail project creates the file LJCDataDetail.exe. This program can be run directly and functions as a test program that displays a sample DataDetailDialog form.

A program reference can be set to LJCDataDetail.exe which allows programatic access to the DataDetailDialog form. This form is used to create and display a DataDetail Dialog from within a program.

The DataDetailDialog form dynamically displays a detail view of the data. This can be used to create or edit data records.
